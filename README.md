# Бот розрахунку ЗП — Інструкція деплою на Railway

## Крок 1 — Підготовка файлів

1. Розпакуй архів
2. Переконайся що в папці є: `bot.py`, `requirements.txt`, `railway.toml`, `lookup.json`

## Крок 2 — GitHub репозиторій

1. Зайди на https://github.com → New repository → назви `salary-bot`
2. Завантаж всі файли з папки в репозиторій (Upload files)

## Крок 3 — Railway

1. Зайди на https://railway.app
2. New Project → Deploy from GitHub repo → вибери `salary-bot`
3. Після деплою зайди в **Variables** і додай:

| Змінна | Значення |
|--------|---------|
| `BOT_TOKEN` | твій токен від @BotFather |
| `ADMIN_IDS` | твій Telegram ID (дізнайся у @userinfobot) |
| `SALARY_PCT` | `20` (% ЗП від прибутку) |
| `BONUS_PCT` | `15` |
| `RATE_MARKUP` | `2` (надбавка до курсу НБУ %) |

4. Railway автоматично перезапустить бота

## Крок 4 — Перевірка

Напиши `/start` своєму боту в Telegram

---

## Як користуватись

**Менеджер:**
1. Надіслати PDF рахунку боту
2. Бот показує всі позиції з закупівельними цінами
3. Ввести дату оплати → бот сам підтягує курс НБУ
4. Вказати складські товари (якщо є) → або написати "ні"
5. `/звіт` → отримати Excel з ЗП

**Керівник:**
- `/адмін` → бачить всіх менеджерів, їх прибуток і ЗП
